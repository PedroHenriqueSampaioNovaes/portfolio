  • Fintech

    Este projeto é uma SPA (single page application) focada na visualização de dados financeiros. O objetivo principal é fornecer uma interface intuitiva para o acompanhamento de vendas, status de pagamentos (pago, processando, falha) e métricas de desempenho. A aplicação consome uma API externa para buscar dados de vendas baseados em intervalos de datas customizáveis. Os dados são processados e apresentados através de gráficos dinâmicos, permitindo uma análise rápida da saúde financeira do negócio.

    Tecnologias usadas:

    React Vite React Router DOM TypeScript Recharts
